# HRMS 技术栈学习 Demo

## 技术栈

- **后端**: Spring Boot 3.3.13 + Spring Data JPA + H2数据库 + JWT认证
- **前端**: Next.js 15 + React 19 + TypeScript

## 功能特性

1. **用户登录** - JWT Token认证
2. **主菜单** - 展示用户信息和功能模块
3. **用户列表** - CRUD操作演示

## 快速启动

### 后端启动

```bash
cd backend
mvn spring-boot:run
```

访问: http://localhost:8080
Swagger文档: http://localhost:8080/swagger-ui.html

### 前端启动

```bash
cd frontend
npm install
npm run dev
```

访问: http://localhost:3000
